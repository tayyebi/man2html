NAME
       getpt - open a new pseudoterminal master

LIBRARY
       Standard C library (libc, -lc)

SYNOPSIS
       #define _GNU_SOURCE             /* See feature_test_macros(7) */
       #include <stdlib.h>

       int getpt(void);

DESCRIPTION
       getpt()  opens  a  new  pseudoterminal device and returns a file descriptor that refers to that device.  It is equivalent to
       opening the pseudoterminal multiplexor device

           open("/dev/ptmx", O_RDWR);

       on Linux systems, though the pseudoterminal multiplexor device is located elsewhere on some systems that use the GNU  C  li‐
       brary.

RETURN VALUE
       getpt() returns an open file descriptor upon successful completion.  Otherwise, it returns -1 and sets errno to indicate the
       error.

ERRORS
       getpt() can fail with various errors described in open(2).

VERSIONS
       getpt() is provided since glibc 2.1.

       getpt() is glibc-specific; use posix_openpt(3) instead.
