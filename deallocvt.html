NAME
       deallocvt - deallocate unused virtual consoles

SYNOPSIS
       deallocvt [option...]  [N ...]

DESCRIPTION
       The  command  deallocvt deallocates kernel memory and data structures for all unused virtual consoles.  If one or more argu‚Äê
       ments N ...  are given, only the corresponding consoles /dev/ttyN are deallocated.

       A virtual console is unused if it is not the foreground console, and no process has it open for reading or writing,  and  no
       text has been selected on its screen.

OPTIONS
       -V, --version
              print program version and exit.

       -h, --help
              show this text and exit.

       chvt(1), openvt(1)
