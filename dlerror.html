NAME
       dlerror - obtain error diagnostic for functions in the dlopen API

LIBRARY
       Dynamic linking library (libdl, -ldl)

SYNOPSIS
       #include <dlfcn.h>

       char *dlerror(void);

DESCRIPTION
       The  dlerror() function returns a human-readable, null-terminated string describing the most recent error that occurred from
       a call to one of the functions in the dlopen API since the last call to dlerror().  The returned string does not  include  a
       trailing newline.

       dlerror() returns NULL if no errors have occurred since initialization or since it was last called.

VERSIONS
       dlerror() is present in glibc 2.0 and later.

ATTRIBUTES

STANDARDS
       POSIX.1-2001.

NOTES
       The  message  returned  by dlerror() may reside in a statically allocated buffer that is overwritten by subsequent dlerror()
       calls.

   History
       This function is part of the dlopen API, derived from SunOS.
