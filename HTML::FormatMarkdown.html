NAME
       HTML::FormatMarkdown - Format HTML as Markdown

VERSION
       version 2.16

SYNOPSIS
           use HTML::FormatMarkdown;

           my $string = HTML::FormatMarkdown->format_file(
               'test.html'
           );

           open my $fh, ">", "test.md" or die "$!\n";
           print $fh $string;
           close $fh;

DESCRIPTION
       HTML::FormatMarkdown is a formatter that outputs Markdown.

       HTML::FormatMarkdown is built on HTML::Formatter and documentation for that module applies to this - especially "new" in
       HTML::Formatter, "format_file" in HTML::Formatter and "format_string" in HTML::Formatter.
