NAME
       arptables-save - dump arptables rules to stdout (nft-based)

SYNOPSIS
       arptables-save [-M modprobe] [-c]

       arptables-save [-V]

DESCRIPTION
       arptables-save  is  used to dump the contents of an ARP Table in easily parseable format to STDOUT. Use I/O-redirection pro‐
       vided by your shell to write to a file.

       -M, --modprobe modprobe_program
              Specify the path to the modprobe program. By default, arptables-save will inspect /proc/sys/kernel/modprobe to deter‐
              mine the executable's path.

       -c, --counters
              Include the current values of all packet and byte counters in the output.

       -V, --version
              Print version information and exit.
