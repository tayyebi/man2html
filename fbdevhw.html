NAME
       fbdevhw - os-specific submodule for framebuffer device access

DESCRIPTION
       fbdevhw  provides  functions for talking to a framebuffer device.  It is os-specific.  It is a submodule used by other video
       drivers.  A fbdevhw module is currently available for linux framebuffer devices.

       fbdev(4) is a non-accelerated driver which runs on top of the fbdevhw module.  fbdevhw can be used  by  other  drivers  too,
       this is usually activated with `Option "UseFBDev"' in the device section.
